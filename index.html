Concepto Descripcion Concepto Descripcion
Tabla en html Es una estructura utilizada para organizar y presentar datos en filas y columnas. Se utiliza comúnmente para mostrar información tabular, como estadísticas, horarios, o listas comparativas. Formulario en html Es una sección de una página web que permite a los usuarios enviar datos al servidor. Los formularios son esenciales para la interacción en línea, como registrarse en un sitio, enviar comentarios o realizar búsquedas.
Etiqueta table Se utiliza para crear tablas, que son estructuras organizadas en filas y columnas. Estas tablas son ideales para presentar datos tabulares de forma clara y accesible. Aquí tienes una descripción más detallada de cómo se utiliza y su estructura básica. Etiqueta from Se utiliza para crear un formulario que permite a los usuarios enviar datos a un servidor. Es fundamental para la interacción en páginas web, ya que se utiliza en registros, inicios de sesión, encuestas y más.

 Etiqueta input Se utiliza para crear campos de entrada en un formulario. Permite a los usuarios ingresar datos, como texto, contraseñas, correos electrónicos, y más. La etiqueta < input > es muy versátil y tiene varios tipos que determinan el tipo de datos que se pueden ingresar.
Etiqueta tr Se utiliza para definir una fila en una tabla. Es un elemento esencial en la estructura de una tabla, ya que agrupa celdas de datos y encabezados dentro de esa fila. Etiqueta label Se utiliza para asociar un texto de etiqueta con un elemento de formulario, como un < input >, < textarea >, < select >, entre otros. Esto mejora la accesibilidad y la usabilidad de los formularios, ya que al hacer clic en la etiqueta, se activa el control correspondiente.

Etiqueta select Se utiliza para crear un menú desplegable que permite al usuario seleccionar una opción de una lista de opciones. Es comúnmente usada en formularios para capturar datos en un formato controlado.
Etiqueta td y th La etiqueta < td > se utiliza para definir una celda en el cuerpo de la tabla que contiene datos. La etiqueta < th> se utiliza para definir una celda de encabezado en una tabla. Los textos dentro de < th > suelen aparecer en negrita y están centrados por defecto. Etiqueta textarea:< /td> Se utiliza para crear un campo de entrada de texto de múltiples líneas, lo que permite a los usuarios ingresar texto de longitud variable, como comentarios o descripciones. Es especialmente útil en formularios donde se requiere un texto más extenso.

 Etiqueta fieldset, legend La etiqueta < fieldset > se utiliza para agrupar controles de formulario relacionados, como campos de entrada y botones, dentro de un cuadro. Tablas avanzadas Se refieren a la utilización de tablas que no solo muestran datos de manera simple, sino que también incorporan características y funcionalidades adicionales para mejorar la presentación, la usabilidad y la interactividad. Etiqueta meter Se utiliza para representar una medida escalar dentro de un rango conocido, como un nivel de habilidad, una cantidad, o un porcentaje. Es útil para mostrar datos cuantitativos en un formato visual, similar a un medidor o una barra de progreso.
Etiqueta thead Se utiliza para agrupar el contenido del encabezado de una tabla. Esto ayuda a organizar y estructurar la tabla, facilitando la comprensión de los datos. La etiqueta < thead > se coloca dentro de una etiqueta < table > y generalmente contiene una o más filas < tr > con celdas de encabezado Etiqueta progress Se utiliza para mostrar el progreso de una tarea en curso, como la descarga de un archivo o el progreso de un formulario.
Etiqueta tfoot Se utiliza para agrupar el contenido del pie de una tabla. Se coloca dentro de una etiqueta < table > y generalmente se utiliza para incluir resúmenes, totales o notas al pie que se relacionan con los datos presentados en el cuerpo de la tabla. Al igual que < thead >, < tfoot> mejora la semántica y la organización de la tabla. Etiqueta datalist Se utiliza para proporcionar una lista de opciones predefinidas que los usuarios pueden seleccionar mientras escriben en un campo de entrada (< input >).
Etiqueta tbody Se utiliza para agrupar el contenido del cuerpo de una tabla. Esta etiqueta ayuda a organizar las filas de datos y proporciona una estructura más clara a la tabla. Al igual que < thead > y < tfoot>, < tbody> mejora la semántica y la accesibilidad de la tabla. Etiqueta summary Se utiliza junto con la etiqueta < details> para crear un cuadro de información que se puede expandir o contraer. El < summary > actúa como el título o encabezado del contenido oculto, que se muestra u oculta cuando el usuario hace clic en él.

Tema 2. Define los elementos que componen la estructuración de una página web:

  1. Contenedor (wrapper)

    2. El contenedor o wrapper es un elemento que envuelve todo el contenido de la página. Su función principal es proporcionar una estructura organizada y facilitar la aplicación de estilos CSS. Generalmente se utiliza para centrar el contenido y establecer anchos máximos, márgenes y rellenos.

  2. Cabecera (header)

    3. La cabecera es la parte superior de la página que generalmente contiene elementos importantes como el logotipo, el título del sitio, y a menudo incluye el menú de navegación principal. Proporciona una introducción visual y funcional a la página.

  3. Contenido (content)

    4. El contenido es la parte principal de la página, donde se presenta la información más relevante, como texto, imágenes, videos y otros elementos multimedia. Este es el área que los visitantes de la página están más interesados en explorar.

  4. Menú (menu)

    5. El menú es una lista de enlaces que permite a los usuarios navegar por diferentes secciones del sitio web. Puede estar ubicado en la cabecera, en un lateral, o incluso en el pie de página. Los menús pueden ser verticales u horizontales y pueden incluir submenús.

  5. Pie (footer)

    6. El pie de página es la sección final de la página web. Suele contener información como derechos de autor, enlaces a políticas de privacidad, términos de servicio, y enlaces adicionales a redes sociales. Es un espacio para información adicional que no se encuentra en otras partes de la página.

  6. Lateral (sidebar)

    7. La barra lateral es un área que se encuentra a un lado del contenido principal y suele contener información adicional, como enlaces a otras secciones, widgets, publicidad, o contenido relacionado. Puede ser fija o desplazarse con el contenido principal.

  7. Etiqueta div

    8. La etiqueta < div > es un contenedor genérico en HTML que se utiliza para agrupar otros elementos. No tiene un significado semántico por sí misma, pero es útil para aplicar estilos CSS y organizar el contenido en secciones. Puede ser utilizada en cualquier parte de la página para dividir y estructurar el contenido de manera flexible.
